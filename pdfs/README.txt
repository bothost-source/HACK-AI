# PDF Upload Instructions

1. Copy your PDF files into this folder
2. Supported: Any text-based PDF (not scanned images)
3. Recommended PDFs for hacking bot:
   - Cybersecurity books
   - Networking guides
   - Coding/programming books
   - Hacking/penetration testing guides
   - Cisco/CCNA materials
   - Forensics books

4. After uploading, run: npm run build-kb
   Or use /reload command in Telegram

5. The bot will automatically chunk and index all PDFs
